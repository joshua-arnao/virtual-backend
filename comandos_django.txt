CLI => Interfaz en línea de comandos (Command Line Interface)
Todos los comandos de django se realizan mediante el comando 'django-admin'
No se puede crear aplicaciones con los nombres de las aplicaciones de Django: auth, contenttypes, message, users, admin, sessions, staticfiles
django-admin startproject NOMBRE_PROYECTO => Creara un nuevo proyecto con los archivos necesarios
cd <carpeta>

python3 manage.py createsuperuser => para dar accesos al panel administrativo
 
python3 manage.py => para ver todos los subcomandos disponibles

python3 manage.py runserver => Levanta el pprojecto de Django

python3 manage.py showmigrations => muestra todas las migraciones que tenemos en nuestro proyecto

python3 manage.py migrate sessions

python3 manage.py migrate => ejecutara la migración pendiente en la bd, adicional se puede colocar el nombrer de la aplicación para que solo se ejecute dicha migración

python3 manage.py makemigrations NOMBRE_APP --name NOMBRE_MIGRACION => creara una nueva migracion de nuestra aplicacion con todos los cambios que se habran hecho en el ORM

python3 manage.py sqlmigrate NOMBRE_MIGRACION

python3 manage.py migrate => crea la tabla

python3 showmigrations <nombre>

python3 manage.py dbshell


python manage.py flush => Elimina toda la información de todas las tablas, reinicia los indices de los autoincementavles y la vuelve a un estado incial
---- Interactive Console ----
python3 manage.py shell
